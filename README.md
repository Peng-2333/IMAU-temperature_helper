# IMAU-temperature_helper 内农大易班体温填报助手
针对内农大易班疫情体温填报所开发的小脚本
--基于AutoX.js
--依赖安卓控件进行识别，兼容性应该还不错

-V1.0 2022.2.24 新手开发的半成品小脚本，没有UI（不会），没有单独打包apk文件，运行依靠 AutoX.js/Hamibot 定时启动运行。
